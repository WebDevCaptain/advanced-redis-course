
Pipelining using a Python client

You may want to run a Redis pipeline programmatically rather than using the command line interface. Let's see how we can do that using Python as a programming language. In my current working directory, I have a python file called pipeline_python.py. This contains code that will allow me to programmatically connect to my Redis server and run multiple operations as a pipeline. Now you need to have Python installed on your local machine so that you can execute this code. I'm going to run python-V and you can see that I'm working with Python version 3.8.8. Your python version shouldn't really matter as long as it's a version 3.7 plus. In order to be able to connect to the Redis server, you need to have the Redis Python package installed on your local machine, so make sure you get it using a pip install redis. This will install the Python library that will allow you to instantiate the Redis server and access it programmatically in Python. Once the installation is complete, let's take a look at the code required to connect to our Redis server and run a pipeline operation. On Line4, I connect to my Redis database server by instantiating the Redis object. The host is localhost, the server is running on my local machine and the port is 6379. Next I instantiate a pipeline object, I call redisobj.pipeline to get a pipeline object allowing me to execute operations in a batch. On Lines 7 through 12, I perform a number of set operations to set values for several different keys name and "a", "b", "p", "q" and "n." These operations will not be executed unless I call pipeline.execute, which I do on line 14. This will cause all of the operations performed so far to be executed in a single batch. Once I've executed the pipeline, I want to check whether these keys have been updated successfully on our Redis server, so I access the original values using redisobj.get This is the code on lines 17 through 22. I access each of the keys that I have added to my Redis server and print out the corresponding values. On lines 24 through 28, I perform a number of update operations on the various keys. Once again using the pipelineobj. I append Jonus to the name, I increment "a" by 1, I increment the value of "b" by 3, I increment "p" by 0.5 and I delete the key, "q". All of these operations will be executed as a pipeline when I call pipeline.execute on Line 32. Once we've executed the updates on the keys that we added, I'm going to print out the updated values and this do on Lines 36 through 41 and that's all this Python code contains. Let's switch back to the terminal window, and I'm now going to run this Python file. And you can see here from the result that things run through successfully. We programmatically created a pipeline to add a number of records to our Redis database. We then update those records using another pipeline and then we queried those records and displayed the results out to screen.
