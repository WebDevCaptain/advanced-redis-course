
Introducing RediStack

In this movie, I'll introduce you to Redis Stack. Redis Stack extends the core capabilities of the Redis OSS with modern data models, data processing tools, and debugging support. With Redis Stack, Redis developers have bundled together the best of Redis technology into an easy to use package. So Redis Stack offers a whole number of extensions, but in this movie, we'll explore just two of Redis Stack's modules in detail, RediSearch and RedisJSON. Both of these are part of Redis Stack. Let's take a quick look at some of the extensions and modules that Redis Stack has to offer. Here are some of the most popular modules that are part of Redis Stack, RedisInsight, RediSearch, RedisJSON, RedisGraph, RedisTimeSeries, and RedisBloom. The first of these, RedisInsight is a visualization tool for understanding and optimizing Redis data. RedisInsight allows you to view data structures visually with a browser tool and group keys according to their namespaces. You can use RedisInsight for CRUD operation on Redis Stack data structures. RedisInsight includes other tools such as the Profiler, which allows you to analyze traffic to Redis in real time, and it also contains an embedded Redis CLI that you can use. Another popular module is RediSearch, which enables complex search queries on your data. This is something that we'll be exploring in more detail in just a bit. RediSearch enables querying secondary indexing, and full text search on data that you store in Redis. RediSearch allows you to index multiple fields in your data, allows you to perform aggregations, exact phrase matching, fuzzy phrase matching, numeric filtering, and semantic search. And really this is just the tip of the iceberg. RediSearch offers incredible search functionality which you ought to explore for yourself even beyond what we cover here in this course. A common format in which your documents may be present is the JSON format, and RedisJSON allows you to store, update and retrieve JSON values from a Redis data store. RedisJSON has full support for the JSON standard. RedisJSON also gives you a JSONpath syntax for selecting/updating elements inside JSON documents. The JSONpath syntax allows you to access and modify top level JSON objects as well as objects that are nested within your top level objects, whether they be arrays or just JSON objects. Your JSON documents are stored as binary data in a tree structure, giving you very fast access to sub elements and RedisJSON also supports typed atomic operations for all JSON types, including arrays. RedisJSON also integrates very nicely with RediSearch to allow indexing and querying of JSON documents. Redis Stack also includes RedisGraph, which is a graph database built on Redis. You can represent graphical structures using nodes and edges. The graph database uses graph blocks under the hood for its sparse adjacency matrix graph representation. The RedisTimeSeries module adds time series data structure to Redis. This enables time based querying, sampling, and secondary indexing for time series data. This allows you to query time series data using start time and end time. You can run aggregated query such as min, max average for any time bucket. And finally RedisBloom supports Bloom filters and other probabilistic data structures on Redis. RedisBloom is typically used to analyze streaming data and very large datasets.
