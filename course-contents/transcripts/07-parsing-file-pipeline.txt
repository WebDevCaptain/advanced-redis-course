
Parsing a text file to run pipeline commands

Let's say you want to store some key value pairs in Redis and you have these key value pairs in a txt file. Let's see how you can programmatically open up this txt file and execute commands using a pipeline. In this movie, we'll be working with two different files and both of these are present in my current working directory. The first file, kv_pairs.txt contains the key value pairs that I want to store in my Redis database. load_kvs.py contains the Python code that I'll use to parse my txt file and run commands using a pipeline. Let's look at kv_pairs.txt, and you can see that we just have key value pairs separated by a comma and every record is on a different line. Let's see how we can parse this file and set these key value pairs in our Redis database programmatically in Python.
